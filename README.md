# Library App
This is a Library App where you can display, add and remove your books. You can use this to keep track of your reading list, and check all your completed books.

## Live Demo Link

[Live Demo Link](https://raw.githack.com/edemagbenyo/library/library/index.html)

=======

## Features

- Display books
- Create book
- Delete books
- Update Read Status

## Built With

- HTML, CSS, Bootstrap, Javascript
- Vscode for coding
- ESLint for Validation

For more information, see the
[*The Odin Project*](https://www.theodinproject.com/courses/javascript/lessons/library).
